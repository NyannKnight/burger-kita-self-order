# 🍔 Burger Kita - Smart Restaurant System
Sistem Informasi Restoran berbasis Web (Self-Order Kiosk, KDS, & POS) yang dibangun untuk tugas Capstone Project Universitas Terbuka.

## 🚀 Fitur Utama
* **Pelanggan:** Self-Order Kiosk, Kalkulasi Keranjang Otomatis.
* **Kasir:** Auto-Refresh Antrean, Cetak Struk Thermal.
* **Dapur:** Kitchen Display System (KDS) terintegrasi.
* **Admin:** Manajemen Menu, Meja, dan Laporan Penjualan (Omzet).

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap.
* **Backend:** PHP Native.
* **Database:** MySQL.

## ⚙️ Cara Instalasi
1. Clone repository ini ke dalam folder `htdocs` (XAMPP).
2. Buat database baru di PHPMyAdmin.
3. Import file `database_burger.sql`.
4. Sesuaikan koneksi database pada file `koneksi.php`.
5. Buka `localhost/nama_folder` di browser Anda.
